Как использовать

Обзор

Наше приложение для установки целей - это инструмент повышения продуктивности, который помогает вам эффективно управлять своими целями и задачами. Приложение разбито на три основных компонента: интерфейс "Goals", интерфейс "Tasks" и интерфейс "Statistics". Кроме того, есть страница "Settings" для управления информацией об учетной записи.

Каждая цель, которую вы устанавливаете, связана с набором задач. Эти задачи можно категоризировать по двум типам: 'ToDo List' и 'Number Type'.

Тип 'ToDo List' содержит список подзадач и отмечается как выполненный, когда все подзадачи выполнены. 'Number Type' включает начальное и целевое значение и отмечается как выполненный, когда достигнуто целевое число.

Можно установить приоритетные задачи для удобного доступа, а также есть встроенная функция AI, которая помогает разбить ваши задачи на более маленькие и управляемые части.

Описание страниц

Goals Page

Это ваша основная панель, где вы можете просмотреть все свои цели, категоризированные по трем папкам: 'In Process' (недостиженные цели), 'Achieved' (достиженные цели) и 'Priority Tasks' (приоритетные задачи).

Tasks Page

При нажатии на определенную цель вы будете перенаправлены на страницу задач, связанных с этой целью. На этой странице есть две папки: 'In Process' (незавершенные задачи) и 'Completed' (завершенные задачи).

Statistics Page

Здесь вы можете просмотреть количество задач, которые вы выполнили за определенный период.

Settings Page

На этой странице вы можете управлять информацией об учетной записи, включая сброс пароля.

Использование приложения

Goals Page

  • Создание цели: нажмите на кнопку "Create New Goal". Во всплывающем модальном окне введите название вашей цели и теги (разделенные запятыми) и нажмите "Create!".
  • Редактирование цели: нажмите на три точки справа от вашей цели и выберите "Edit" для изменения названия и тегов цели.
  • Удаление цели: нажмите на три точки рядом с вашей целью и выберите "Delete" для удаления вашей цели и связанных с ней задач.
  • Достижение цели: нажмите на три точки рядом с вашей целью и выберите "Achieved" для перемещения ее в папку 'Achieved'.
  • Поиск целей: используйте поле поиска в верхней части главной страницы для поиска целей по тегам или названиям.

Tasks Page

  • Создание задачи: нажмите "Create New Task", напишите название задачи, выберите тип задачи (ToDo List или Number Type), введите содержимое и нажмите "Create!".
  • Установка прогресса задачи: нажмите на задачу, чтобы обновить ее прогресс.
  • Редактирование задачи: нажмите на три точки рядом с задачей и выберите "Edit" для изменения ее содержимого.
  • Удаление задачи: нажмите на три точки рядом с задачей и выберите "Delete" для ее удаления.
  • Приоритет задачи: нажмите на три точки рядом с задачей и выберите "Priority", чтобы установить ее как приоритетную задачу, которая будет отображаться на вкладке 'Priority Tasks' на главной странице.

Типы задач

ToDo List Type

Этот тип задачи лучше всего подходит для целей, которые требуют нескольких этапов или стадий для завершения. Он позволяет вам разбить большую задачу на несколько подзадач, делая ее более управляемой и менее подавляющей. Когда все подзадачи выполнены, задача ToDo List отмечается как выполненная.

Пример: если ваша цель - написать эссе, вы можете создать задачу ToDo List с подзадачами, такими как "Choose a topic", "Research the topic", "Write a draft", "Revise the draft" и "Proofread and finalize". Каждую из этих подзадач можно отметить как выполненную, когда вы их завершите.

Чтобы добавить новую задачу ToDo List, нажмите "Create New Task", выберите тип 'ToDo List', напишите каждую подзадачу, разделенную запятыми, в поле содержимого и нажмите "Create!". Также есть функция использования AI, которая может помочь разбить вашу задачу на более мелкие подзадачи. Чтобы использовать это, нажмите кнопку "Use AI", заполните содержимое, описывающее вашу задачу, и нажмите "Break Task".

Number Type

Этот тип задачи идеально подходит для целей, которые включают достижение определенного числового целевого значения. Вы определяете начальное и целевое значение, и задача отмечается как выполненная, когда достигнуто целевое число.

Пример: если ваша цель - сделать 100 отжиманий за день, вы можете создать задачу Number Type с начальным значением 0 и целевым значением 100. Каждый раз, когда вы делаете отжимание, вы можете обновить текущее значение, и задача будет отмечена как выполненная, когда вы достигнете 100.

Чтобы добавить новую задачу Number Type, нажмите "Create New Task", выберите 'Number Type', установите начальное и целевое значение в поле содержимого и нажмите "Create!".

Используя эти различные типы задач, вы можете настроить вашу установку целей и отслеживание для любого типа цели, которую вы можете иметь, будь то задача или число.

Statistics Page

Страница статистики предоставляет визуальное представление количества задач, которые вы выполнили за выбранный период.

Settings Page

На странице настроек вы можете сбросить свой пароль.
